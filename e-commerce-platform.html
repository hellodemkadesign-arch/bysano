E-commerce platform
for a local clothing brand

A proprietary branded marketplace with the service level of major fashion platforms

We created a web and mobile application that replicates familiar marketplace scenarios in user experience and logic, but is fully focused on one brand.

Deadlines
6 weeks
Project Type
Application Design
Scope
Telegram Mini Apps UX/UI design
Technologies
Flutter, FlutterFlow, YClients, Supabase (PostgreSQL)

Why did we take on this project?

It was necessary to create not just a sales showcase, but a strategic growth point for the brand.

  • The project involved creating a full-fledged branded service, not a template online store.
  • We reduced dependence on marketplaces and commissions.
  • The project format allowed us to apply our marketplace and e-commerce experience.

Task

Automate clothing brand sales and create a proprietary channel with the service level of marketplaces.

  • Launch a proprietary clothing sales app, independent of Wildberries and other marketplaces.
  • Ensure a familiar UX for customers: search, filters, cart, delivery, and returns.
  • Synchronize orders and inventory with the already used 1C.
  • Establish direct communication with customers and increase brand loyalty.

What did we do? Key decisions

  • Created a branded marketplace with a familiar user experience.
  • Adapted Lamoda-style UX patterns to the client brand.
  • Integrated delivery and returns through CDEK via API.
  • Set up 1C integration for up-to-date inventory and orders.
  • Added banners and stories to engage users and increase conversion.

Catalog and product search

Users can search for products, filter by type, color, and size, view product details, and add items to the cart.

Cart and checkout

From the cart, users can directly proceed to checkout by providing a delivery address. CDEK handles the delivery process through API integration.

Orders and returns

If a product is unsuitable, customers can start a return through a CDEK pickup point or request a courier pickup at home.

Banners and stories

We added banners and stories so the brand could communicate with customers and increase purchases from the homepage.
